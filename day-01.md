# CSE English Vocabulary - Day 01

| No. | Word           | English Meaning                           | বাংলা অর্থ                             |
| --- | -------------- | ----------------------------------------- | -------------------------------------- |
| 1   | Algorithm      | Step-by-step procedure to solve a problem | সমস্যা সমাধানের ধাপসমূহ                |
| 2   | Function       | Reusable block of code                    | নির্দিষ্ট কাজের জন্য কোড ব্লক          |
| 3   | Variable       | Named storage location for data           | ডেটা সংরক্ষণের চলক                     |
| 4   | Parameter      | Input defined in a function               | ফাংশনের ইনপুট নির্ধারণকারী             |
| 5   | Argument       | Actual value passed to a function         | ফাংশনে পাঠানো প্রকৃত মান               |
| 6   | Return         | Send a result back from a function        | ফলাফল ফেরত দেওয়া                      |
| 7   | Compile        | Convert source code into executable code  | সোর্স কোডকে এক্সিকিউটেবল কোডে রূপান্তর |
| 8   | Execute        | Run a program                             | প্রোগ্রাম চালানো                       |
| 9   | Debug          | Find and fix errors                       | ত্রুটি খুঁজে বের করে ঠিক করা           |
| 10  | Syntax         | Rules of a programming language           | ভাষার লেখার নিয়ম                      |
| 11  | Logic          | Reasoning behind a solution               | যুক্তি বা চিন্তাধারা                   |
| 12  | Data           | Information or facts                      | তথ্য                                   |
| 13  | Structure      | Organization or arrangement               | গঠন বা কাঠামো                          |
| 14  | Array          | Collection of similar data items          | একই ধরনের ডেটার সংগ্রহ                 |
| 15  | Index          | Position of an item in a collection       | অবস্থান নম্বর                          |
| 16  | Loop           | Repetition construct                      | পুনরাবৃত্তিমূলক কাজ                    |
| 17  | Condition      | Rule used for decision making             | শর্ত                                   |
| 18  | Iterate        | Repeat through items                      | বারবার অতিক্রম করা                     |
| 19  | Declare        | Create or define a variable               | ঘোষণা করা                              |
| 20  | Initialize     | Assign an initial value                   | প্রাথমিক মান নির্ধারণ করা              |
| 21  | Instance       | Specific object created from a class      | ক্লাস থেকে তৈরি নির্দিষ্ট অবজেক্ট      |
| 22  | Object         | Entity containing data and methods        | ডেটা ও মেথড সমন্বিত সত্তা              |
| 23  | Class          | Blueprint for creating objects            | অবজেক্ট তৈরির নকশা                     |
| 24  | Method         | Function inside a class                   | ক্লাসের ভিতরের ফাংশন                   |
| 25  | Attribute      | Property of an object                     | বৈশিষ্ট্য                              |
| 26  | Inherit        | Acquire features from a parent class      | উত্তরাধিকারসূত্রে গ্রহণ করা            |
| 27  | Encapsulation  | Hiding internal details                   | তথ্য আড়াল করে রাখা                    |
| 28  | Abstraction    | Showing only essential details            | অপ্রয়োজনীয় তথ্য লুকিয়ে রাখা         |
| 29  | Interface      | Contract for interaction                  | যোগাযোগের নিয়ম                        |
| 30  | Framework      | Structure for software development        | সফটওয়্যার উন্নয়নের কাঠামো            |
| 31  | Library        | Collection of reusable code               | পুনঃব্যবহারযোগ্য কোডের সংগ্রহ          |
| 32  | Dependency     | Required external component               | নির্ভরশীল উপাদান                       |
| 33  | Repository     | Storage location for a project            | প্রজেক্ট সংরক্ষণের স্থান               |
| 34  | Commit         | Save changes in Git                       | Git-এ পরিবর্তন সংরক্ষণ                 |
| 35  | Branch         | Separate line of development              | উন্নয়নের আলাদা শাখা                   |
| 36  | Merge          | Combine branches together                 | একত্রিত করা                            |
| 37  | Clone          | Copy a repository                         | কপি তৈরি করা                           |
| 38  | Deploy         | Publish an application                    | সার্ভারে প্রকাশ করা                    |
| 39  | Server         | Computer providing services               | সেবা প্রদানকারী কম্পিউটার              |
| 40  | Client         | Computer receiving services               | সেবা গ্রহণকারী                         |
| 41  | Request        | Ask for data or service                   | অনুরোধ                                 |
| 42  | Response       | Reply to a request                        | উত্তর                                  |
| 43  | Database       | Organized data storage system             | তথ্যভান্ডার                            |
| 44  | Query          | Request information from a database       | তথ্য অনুসন্ধানের নির্দেশ               |
| 45  | Authentication | Verification of identity                  | পরিচয় যাচাই                           |
| 46  | Authorization  | Verification of permission                | অনুমতি যাচাই                           |
| 47  | Secure         | Protected from threats                    | নিরাপদ                                 |
| 48  | Optimize       | Improve efficiency or performance         | কর্মক্ষমতা উন্নত করা                   |
| 49  | Scalable       | Able to grow efficiently                  | সহজে সম্প্রসারণযোগ্য                   |
| 50  | Maintainable   | Easy to update and manage                 | সহজে রক্ষণাবেক্ষণযোগ্য                 |
